# 统一日历事件表单｜增量 Checklist

## 0. 文档范围

本 Checklist 仅用于实现统一事件表单及其默认值变化。

以下内容不在本 Checklist 中重复拆分：

- Calendar 页面整体开发
- 月视图和列表视图
- 日期识别算法
- Calendar CRUD 全部接口
- 数据库整体建模
- 来源删除关系
- 权限系统
- 外部日历同步

上述内容继续使用原《知识库日历功能 Checklist》。
